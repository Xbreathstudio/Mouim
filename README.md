# Mouim 💬

**A modern chat platform by XbreathStudio**

![Version](https://img.shields.io/badge/version-1.1.10--alpha-orange)
![Status](https://img.shields.io/badge/status-alpha-red)

Mouim is a modern real-time chat application focused on speed, simplicity, and user privacy.

Built from scratch with 0 budget, Mouim evolved through multiple redesigns and rebuilds to become a fully functional communication platform.

- 💬 Real-time messaging

- 🌙 Day/Night theme (Auto supported)
